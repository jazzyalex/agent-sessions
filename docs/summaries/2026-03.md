## March 2026

- Agent Cockpit: Idle HUD rows now keep the session display name visible, move activity recency into a compact elapsed column (for example `16s`), show active dots as solid green, and pulse idle amber dots with a larger size after 10 minutes idle.
- Session list (Unified): `CLI Agent` live-state visuals now match Agent Cockpit, with solid green active dots, pulsing amber idle dots that grow slightly after 10 minutes idle, and dimmed idle source cells.
- Agent Cockpit + Session list (Unified): Idle amber status dots now brighten at pulse peak, improving visibility of the pulsing animation.
- Agent Cockpit: Active/idle filtering now uses three single-select pills (`All N`, `Active N`, `Idle N`) with amber `Active` and green `Idle` accents, keeping filter state clear while matching cockpit pill styling.
- Agent Cockpit: Compact mode now hides titlebar controls/chrome, stops re-forcing compact height after manual resize, and correctly accounts for grouped/divider layout units so rows are not hidden.
- Agent Cockpit: Grouped compact mode now adds bottom breathing room to keep the last row fully visible, and compact→full transitions now restore a safe expanded height so grouped rows are not truncated.
- Agent Cockpit: Compact sizing now also updates correctly when live mode is disabled and keeps disabled callout content fully visible without clipping.
- Agent Cockpit: Row typography/colors for agent, project, and session names now match the main Session list style.
- Agent Cockpit: Mouse and keyboard navigation now share one gray selected-row style, and non-selected hover row fill was removed to keep focus/selection state unambiguous.
- Agent Cockpit: Compact mode now keeps filter pills in the toolbar, removes the row keyboard-shortcut badge column, and trims compact sizing so no extra blank bands appear above the list or below the last visible row.
- Agent Cockpit: Full mode now wraps long session names to two lines, while compact mode keeps names on one line and auto-resizes window height to fit visible sessions (up to 8) with title text hidden.
- Agent Cockpit: Replaced the `Agent Cockpit` window UI with the new HUD design (floating panel style, active/idle chips, inline filtering, grouped view, compact mode, pin mode, and row keyboard shortcuts), while keeping Legacy Cockpit available.
- Agent Cockpit: Window title now shows the currently displayed session count (`Agent Cockpit (N)`), removed the in-header `AGENT COCKPIT` label and footer status row, and removed the focused-window blue focus-ring box from the HUD list area.
- Menu/Cockpit windows: Removed `Legacy Cockpit` from the `View` menu, leaving `Agent Cockpit` as the only cockpit entry in the main menu.
- Menu/Cockpit windows: Removed `Legacy Cockpit` from the macOS `Window` menu while keeping the legacy scene code path in place.
- Agent Cockpit: This release keeps backend behavior parity with Legacy Cockpit and intentionally hides the branch column in all HUD modes.
- Menu/Cockpit windows: `View > Cockpit` was split into `Agent Cockpit` and `Legacy Cockpit`; both are single-instance windows.
- Shortcuts/filters: `⌘⌥⇧C` now opens `Agent Cockpit`, while `Legacy Cockpit` has no keyboard shortcut and defaults its filter mode to `Live`.
- Claude usage tracking: Hardened tmux probe cleanup by removing managed probe sockets at script exit and shifting orphan cleanup to socket-liveness classification, preventing stale-label churn and large CPU spikes when Claude tracking is enabled.
- Sessions (Gemini): Discovery now accepts named project folders under `~/.gemini/tmp` and continues to support both `chats/session-*.json` and direct `session-*.json` layouts, restoring pickup for newer Gemini CLI directory naming.
- Sessions (Agent Cockpit/Codex live): Pinned Agent Cockpit now refreshes faster while the app is backgrounded (3s cadence), so session rows and live status stay fresher without requiring app focus.
- Sessions (Agent Cockpit/Codex live): Foreground-return live-state probing now ramps in bounded batches and reuses a single iTerm session-list fetch across Codex+Claude, reducing short CPU spikes when switching back to Cockpit.
- Sessions (Agent Cockpit/Codex live-state): Codex ambiguous non-prompt iTerm tails now use log-write heuristic fallback instead of immediately forcing idle, reducing false-idle flips while preserving prompt-based idle detection.
- Sessions (Agent Cockpit/Codex live-state): Background iTerm probing is now deferred unless the app is foregrounded or a pinned cockpit is visible, reducing background probe load for non-pinned windows.
