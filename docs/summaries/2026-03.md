## March 2026

- Agent Cockpit: Replaced the `Agent Cockpit` window UI with the new HUD design (floating panel style, active/idle chips, inline filtering, grouped view, compact mode, pin mode, and row keyboard shortcuts), while keeping Legacy Cockpit available.
- Agent Cockpit: Window title now shows the currently displayed session count (`Agent Cockpit (N)`), removed the in-header `AGENT COCKPIT` label and footer status row, and removed the focused-window blue focus-ring box from the HUD list area.
- Menu/Cockpit windows: Removed `Legacy Cockpit` from the `View` menu, leaving `Agent Cockpit` as the only cockpit entry in the main menu.
- Menu/Cockpit windows: Removed `Legacy Cockpit` from the macOS `Window` menu while keeping the legacy scene code path in place.
- Agent Cockpit: This release keeps backend behavior parity with Legacy Cockpit and intentionally hides the branch column in all HUD modes.
- Menu/Cockpit windows: `View > Cockpit` was split into `Agent Cockpit` and `Legacy Cockpit`; both are single-instance windows.
- Shortcuts/filters: `⌘⌥⇧C` now opens `Agent Cockpit`, while `Legacy Cockpit` has no keyboard shortcut and defaults its filter mode to `Live`.
- Claude usage tracking: Hardened tmux probe cleanup by removing managed probe sockets at script exit and shifting orphan cleanup to socket-liveness classification, preventing stale-label churn and large CPU spikes when Claude tracking is enabled.
- Sessions (Gemini): Discovery now accepts named project folders under `~/.gemini/tmp` and continues to support both `chats/session-*.json` and direct `session-*.json` layouts, restoring pickup for newer Gemini CLI directory naming.
- Sessions (Agent Cockpit/Codex live): Pinned Agent Cockpit now refreshes faster while the app is backgrounded (3s cadence), so session rows and live status stay fresher without requiring app focus.
- Sessions (Agent Cockpit/Codex live): Foreground-return live-state probing now ramps in bounded batches and reuses a single iTerm session-list fetch across Codex+Claude, reducing short CPU spikes when switching back to Cockpit.
