## PR submission checklist

- [ ] Fork the target repo
- [ ] Add entry in the correct alphabetical position within the relevant section
- [ ] Follow the existing entry format exactly (spacing, link format, badges)
- [ ] PR description is 2–3 sentences, factual, no superlatives
- [ ] Check if the list has a contribution guide (CONTRIBUTING.md) and follow it
- [ ] Do not submit to all lists on the same day — spread over T-3 to T-1
