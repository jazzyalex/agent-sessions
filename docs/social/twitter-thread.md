## Post 3 — The workflow

My favorite workflow:

1. Open Agent Sessions
2. Search "migration" across all my agents
3. Find the session from last week where I worked out the strategy
4. Right-click → Copy Resume Command
5. Paste into terminal, keep going

No re-explaining. No lost context.
