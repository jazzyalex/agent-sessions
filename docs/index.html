Agent Sessions

macOS • Open Source

Unified Session Browser for Codex & Claude

Agent Sessions 2.0 brings Codex CLI and Claude Code together in a single, powerful interface. Stop grepping through scattered JSONL files—get instant search, dual usage tracking, and one-click resume for both tools. Local-first, private, and designed for developers.

Download Agent Sessions 2.0 View on GitHub

Install via Homebrew Tap

brew tap jazzyalex/agent-sessions
brew install --cask agent-sessions

Unified Interface

Browse Codex and Claude sessions side-by-side. Toggle between sources (Both/Codex/Claude) with strict filtering and unified search.

Advanced Search v2

Two-phase incremental pipeline with progress tracking. Cancel instantly, search small files first, defer large sessions. Full-text with repo/path operators.

Resume Anywhere

One-click launch for Codex CLI or Claude Code in Terminal/iTerm. Automatic working directory resolution, trust confirmations, and context restoration.

Dual Usage Tracking

Separate 5-hour and weekly limits for Codex and Claude. Menu bar widget with color-coded thresholds and reset time displays.

Lightning Fast

Lazy hydration for 10+ MB sessions, off-main parsing, metadata-first indexing. Handles 1000+ sessions without UI freezes.

Local & Private

All processing on your Mac. Reads ~/.codex/sessions and ~/.claude/sessions locally. No cloud uploads or telemetry.
