Agent Sessions
• macOS • Open Source

Session Browser for Codex CLI, Claude Code & Gemini CLI

Agent Sessions 2 brings Codex CLI, Claude Code, and Gemini CLI together in a single, powerful interface. Look up any past session — even the ancient ones /resume can't show — or browse visually to find that lost perfect prompt or code snippet, then instantly copy or resume what you need.

local-first, open source, and built for terminal vibe warriors.

Download Agent Sessions 2.5.2 View on GitHub Follow on X

Install with Homebrew Tap

brew tap jazzyalex/agent-sessions
brew install --cask agent-sessions

Unified Interface v2

Browse Codex CLI, Claude Code, and Gemini CLI side-by-side. Toggle between sources with strict filtering and unified search.

★ Star important sessions for quick access — one click in the toolbar brings up your favorites.

Unified Search v2

One unified search across Codex, Claude Code, and Gemini CLI sessions. Find any snippet or prompt instantly — no matter which agent or terminal it came from. Smart sorting, instant cancel, full-text search with project filters.

Instant Resume & Re-use

Instantly reopen any Codex or Claude Code session in Terminal/iTerm2. When /resume fails or you only need a snippet — browse visually, copy the snippet, and paste it into a new terminal session or chat.

Usage Tracking

Always-on 5-hour and weekly limits for Codex CLI and Claude Code. Menu bar or in-app strip indicators with thresholds and reset time.

Analytics & Git Inspector

Session stats, agent breakdowns, and time-of-day heatmaps show where your coding time actually goes. Git Inspector catches branch changes and uncommitted work before you resume sessions.

Local, Private & Safe

All processing on your Mac. Reads ~/.codex/sessions, ~/.claude/sessions and ~/.gemini/tmp. No cloud uploads or telemetry. Read only – your sessions are always safe.
