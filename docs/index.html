Agent Sessions
• macOS • Open Source

Session Browser for Codex CLI & Claude Code

Agent Sessions 2 brings Codex CLI and Claude Code together in a single, powerful interface. Look up any past session — even the ancient ones /resume can't show — or browse visually to find that lost perfect prompt or code snippet, then instantly copy or resume what you need.

local-first, open source, and built for terminal vibe warriors.

Download Agent Sessions 2.2.1 View on GitHub Follow on X (Twitter)

Install with Homebrew Tap

brew tap jazzyalex/agent-sessions
brew install --cask agent-sessions

Unified Interface v2

Browse Codex CLI and Claude Code sessions side-by-side. Toggle between sources (Both/Codex/Claude) with strict filtering and unified search.

Unified Search v2

One unified search across Codex CLI and Claude Code sessions. Find any snippet or prompt instantly — no matter which agent or terminal it came from. Smart sorting, instant cancel, full-text search with project filters.

Instant Resume & Re-use

Instantly reopen any Codex or Claude Code session in Terminal/iTerm2. When /resume fails or you only need a snippet — browse visually, copy the snippet, and paste it into a new terminal session or chat.

Usage Tracking

Always-on 5-hour and weekly limits for Codex CLI and Claude Code. Menu bar or in-app strip indicators with thresholds and reset time.

Advanced Analytics

coming soon

Cross-analyze your full Codex CLI and Claude Code history to spot trends, habits, and overlaps across projects. Track usage, prompt styles, and performance patterns to understand how your workflow evolves over time.

Local, Private & Safe

All processing on your Mac. Reads ~/.codex/sessions and ~/.claude/sessions locally. No cloud uploads or telemetry. Read only – your sessions are always safe.
