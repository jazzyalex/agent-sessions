Agent Sessions

macOS • Open Source

MacOS viewer for Coding CLI agents sessions

Agent Sessions helps developers search, filter, and explore plain-text transcripts of Coding CLI agents sessions (Codex, Claude). It’s local-first, private, and designed with developers in mind.

⬇ Download for macOS View on GitHub

Install via Homebrew Tap

brew tap jazzyalex/agent-sessions
brew install --cask agent-sessions

Plain Transcript

Readable transcript viewer with no truncation. Copy & paste exactly what you see.

Search & Filters

Sort and filter by Modified, Msgs, Title, Repo. Quickly find the session you need.

Repo Awareness

Repo column with tooltip showing the full directory path for context.

Resume (Experimental)

This feature is still in development. Attempt to resume sessions in Terminal using upstream flags. Falls back safely if unsupported.

Local-only

Runs on your Mac. Reads ~/.codex/sessions. No network or cloud required.
